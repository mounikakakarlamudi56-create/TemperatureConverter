# 🌡️ Temperature Converter System

A simple Java console application that converts temperatures between Celsius, Fahrenheit, and Kelvin using a menu-driven interface.

## Features
- Celsius ↔ Fahrenheit
- Celsius ↔ Kelvin
- Fahrenheit ↔ Kelvin
- Menu-driven interface
- Input validation
- Multiple conversions

## Technologies
- Java
- VS Code
- JDK

## How to Run

```bash
javac TemperatureConverter.java
java TemperatureConverter
```
